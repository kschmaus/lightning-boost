# Contributing

Thanks for your interest in contributing to ngboost-lightning!

For development setup, testing, linting, and project structure details, see the
full contributing guide:

**<https://kschmaus.github.io/ngboost-lightning/contributing/>**

## Quick Start

```bash
git clone https://github.com/kschmaus/ngboost-lightning.git
cd ngboost-lightning
uv sync --group dev
uv run pytest
```
